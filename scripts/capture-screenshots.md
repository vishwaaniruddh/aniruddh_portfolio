## Screenshot Settings:
- Resolution: 1920x1080 (desktop view)
- Format: PNG or WebP
- Aspect Ratio: 16:9 or 4:3
- Focus: Above the fold content

## Tools You Can Use:
1. **Browser DevTools** (F12) - Device toolbar for consistent sizing
2. **Full Page Screenshot Extensions**:
   - GoFullPage (Chrome)
   - Awesome Screenshot
   - Fireshot
3. **Online Tools**:
   - https://www.screenshotmachine.com/
   - https://www.websiteplanet.com/webtools/screenshot/
4. **Command Line** (if you have Node.js):
   ```bash
   npx capture-website https://srishringarr.com/ --output=public/projects/srishringarr.png --width=1920 --height=1080
   ```
