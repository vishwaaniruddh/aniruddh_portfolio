# Pre-Deployment Checklist

## ✅ Before Deploying

- [ ] All personal information updated (name, email, phone, social links)
- [ ] All project screenshots added to `public/projects/`
- [ ] Test the build locally: `npm run build`
- [ ] Check for any console errors: `npm run dev`
- [ ] Verify all links work (social media, project links, email, phone)
- [ ] Test responsive design on mobile devices
- [ ] Remove any placeholder content
- [ ] Update README.md with your information
- [ ] Commit all changes to Git

## 🚀 Quick Deploy Commands

```bash
# 1. Build locally to test
npm run build

# 2. If build succeeds, commit and push
git add .
git commit -m "Ready for production deployment"
git push origin main

# 3. Deploy to Netlify (if using CLI)
netlify deploy --prod
```
